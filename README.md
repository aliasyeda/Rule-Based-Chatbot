# 🤖 Rule-Based Chatbot


A simple conversational chatbot that uses pattern matching to respond to user inputs.

## Features
- Responds to greetings, questions about time/date, and small talk
- Maintains conversation history
- Can add new responses during conversation
- Logs all conversations to a JSON file

## How to Run
1. Make sure you have Python installed
2. Run the chatbot:
python chatbot.py

text
3. Give your bot a name (or press Enter for default)
4. Start chatting!

## Commands
- Type `exit` to quit
- Type `history` to see conversation history
- Type `add` to teach the bot new responses

## Example Conversation
You: Hello
Bot: Hello! How can I help you today?

You: What time is it?
Bot: The current time is 14:30

You: exit
Bot: Goodbye! Have a great day!



## Files
- `chatbot.py` - Main program
- `conversation_log.json` - Saved conversations (created when you run the program)
